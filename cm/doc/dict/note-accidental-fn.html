[Function]
(note-accidental note)

Returns the accidental of a note or note class. 

cm> (note-accidental 'dff5)
ff
cm> (note-accidental 'd5)
#f
cm> (note-accidental 'dn5)
n
cm> (note-accidental 'ds)
s
cm>
