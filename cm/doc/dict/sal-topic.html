SAL

SAL is an alternate (non-Lisp) syntax for working with Common Music. It implements an easy-to-learn command language for algorithmic music composition that does not require any familiarity with Lisp notation or Lisp evaluation rules. (The name SAL stands for Simple Algorithmic Language, or Secretly Another Lisp, as you wish.)

Sal consists of two software components: a lexer and parser loaded into Common Music and an Emacs editing mode (sal-mode) that supports developing SAL programs and executing SAL commands. (See the the SAL installation instructions for information on how use SAL in Emacs)

As a way of introducing SAL, here are some equivalent statements using SAL and Lisp notation:

Expressions:

SAL: 
print 2 + 3 * 4 - 1
Lisp: 
(print (- (+ 2 (* 3 4)) 1))

Variable Assignment:

SAL: 
set a = random(10), b += a, c &= b
Lisp: 
(setf a (random 10) b (+ b a) c (append c (list b)))

Global variable definition:

SAL:
define variable 2pi = 2 * pi
Lisp: 
(defparameter 2pi (* 2 pi))

Function definition:

SAL: 
define function rankey (low, high)
  begin
    with range = high - low
    return low + random(range)
  end
Lisp: 
(defun rankey (low high)
  (let ((range (- high low)))
    (+ low (random range))))

Musical process definition:

SAL: 
define process foo (n, r)
  run repeat n
    for k = rankey(60, 80)
    output #a[midi time: now() keynum: k]
    wait r
  end
Lisp: 
(defun foo (n r)
  (process repeat n
           for k = (rankey 60 80)
           output (new midi :time (now) :keynum k)
           wait r))

A SAL program consists of top-level commands, language statements and symbolic expressions (sexprs). A sexpr evaluates to a value. A statement is a language construct. A command is a statement available at the "top-level" that accomplishes interactive tasks such as computing scores, defining functions and processes, loading files, and so on.

Statements

[Command]
set var op expr {, var op expr}*

Assigns values to one or more comma delimited variables. Each var is set to the value of expr according to the assignment operator op:

operatormeaning
= sets var to the value of expr
+= increments var by the value of expr
*= scales var by the value of expr
&= appends value of expr to the list in var
@= prepends value of expr to the list in var
