[Class]
midi

Generates MIDI note on and off pairs from a more general representation of keynum, duration and amplitude.

midi supports the following slot initializations:

:time number
The start time of object in score.
:channel integer
A MIDI channel number, defaults to 0.
:keynum {keynum | note}
A MIDI keynum or note name in the standard chromatic scale. The floating point keynums are rounded to their closest integer keynum unless channel tuning is enabled in the destination midi stream.
:duration number
The duration of the note in seconds.
:amplitude number
A floating-point logical amplitude 0.0 < 1.0, or an integer velocity 0 to 127. Defaults to 64.

Examples:

Example 1. Creating a MIDI note. 

;;; a quiet middle C on channel 3

(new midi :time 0 :duration .5 :channel 3
     :keynum 60 :amplitude .1)

;;; keynum specified as a note name

(new midi :time 4 :keynum 'cs4 :duration 4)

Example 2. Generating untempered MIDI notes. 

(define (harms fund low high dur)
  ;; play harmonics low to high starting on fund
  (let ((f (hertz fund)))
    (process for i from low to high
             output
             (new midi :time (now)
                  ;; convert hertz to floating point keynum
                  :keynum (keynum (* f (/ i low)) :hz true)
                  :duration dur :amplitude .5)
             wait (+ dur .1))))

;;; set channel tuning of midi file to 48 steps per
;;; octave, or 25 cents (1/8 tone) per step. this quantization
;;; size uses the first four channels (0-3) of the synth.

(io "test.mid" :channel-tuning 48)

;;; play partials 4-16 staring on c3

(events (harms 'c3 4 16 .5) "test.mid")

;;; cancel channel-tuning

(io "test.mid" :channel-tuning false)
