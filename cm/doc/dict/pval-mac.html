[Macro]
(pval expr)

Creates a pval object to evaluate expr each time next encounters it in a pattern. The name pval stands for "pattern value". 

cm> (define x (new cycle 
                       :of (list 'a 
                                 (pval (between 1000 5000))
                                 'b)))

cm> (next x 12)
(a 2760 b a 3589 b a 1175 b a 1515 b)
cm>
