[Class]
midi-program-change

A MIDI control change or mode change event. See General MIDI Level 1 Sound Set for a table listing General MIDI (GM) program change messages.

midi-program-change supports the following slot initializations:

:time number
Sets the start time of object in score.
:channel integer
A MIDI channel number. Defaults to 0.
:program integer
A MIDI program change number from 0 to 127. Note that if your MIDI documentation indexes program changes starting with 1 you must subtract 1 when specifying the :program value. For example, Grand Piano is 0, not 1.

Examples:

Example 1. Creating MIDI program changes. 

;;; A GM 'acoustic-bass' program change.

(new midi-program-change :time 0 :channel 3 :program 32)
