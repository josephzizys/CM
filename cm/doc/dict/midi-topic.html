MIDI

Common Music supports reading and writing all types of MIDI messages to MIDI files and to the Midishare real time operating system. There are two levels of MIDI support. A high level interface defines Goops/CLOS classes and methods for generating music to MIDI score files and to MidiShare clients in faster-than-real-time mode. A low level interface defines foreign functions and events for working directly with Midishare in an "interactive", or real-time mode.

High level (non-real time) MIDI support

High level MIDI support provides class definitions for MIDI input and output services and for MIDI event messaging.
