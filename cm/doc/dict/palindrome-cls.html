palindrome

Enumerates data in palindromic order. Subpatterns continue to produce their own ordering.

palindrome supports the following slot initializations:

:elide {false | true | :first | :last | pattern}
If :elide is false (the default) then the first and last value in the pattern are directly repeated as the pattern reverses direction. If the value is :first then the first element is elided, otherwise if the value is :last then the last element is elided. If elide is true then neither the first nor last elements are directly repeated when the pattern changes direction. 
cm> (define x (new palindrome :of '(a b c d)))

cm> (next x 16)
(a b c d d c b a a b c d d c b a)
cm> (define x (new palindrome :of '(a b c d) 
                       :elide true))

cm> (next x 12)
(a b c d c b a b c d c b a b c d)
cm>
