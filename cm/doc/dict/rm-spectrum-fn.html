[Function]
(rm-spectrum spec1 spec2 {keyword value}*)

Performs ring modulation on the input spectra spec1 and spec2 and returns the resulting spectrum as a list of hertz values, key numbers or note names. Partials in the output spectrum are calculated as the difference and sum tones between pairwise combinations of the partials in the input spectra. Values in the input spectra can be specified as hertz values, key numbers or note names but both lists must be of the same type. Each input spectrum can be list of values or a single value.

rm-spectrum supports the keyword arguments:

:spectrum { :note | :keynum | :hertz | false}
Determines what type of values are returned in the output spectrum. The default value is false, which means that values in the output spectrum will be the same type as the input spectra.
:hz boolean
If true then values in the input spectra are Hertz values. The default value is false.
:remove-duplicates boolean
If true then the output spectrum will not contain any duplicate frequencies.

Examples:

Example 1. Ring modulated spectra. 

(rm-spectrum 'd5 'e5)
⇒ (d2 ef6)
(rm-spectrum 'a4 '(c4 ds5) :spectrum ':keynum)
⇒ (53.36887 53.741356 77.07837 84.25864)
(rm-spectrum '(300 400) 550 :hz #t)
⇒ (150 250 850 950)
(rm-spectrum '(300 400) 550 :hz #t :spectrum ':note)
⇒ (d3 b3 af5 bf5)
(rm-spectrum '(c4 e4) '(c4 e4 g4))
⇒ (b1 cs2 cs2 c3 d5 d5 e5 fs5)
(rm-spectrum '(c4 e4) '(c4 e4 g4) :remove-duplicates #t)
⇒ (b1 cs2 c3 d5 e5 fs5)

Example 2. Ring modulating random scale selections. 

(define (random-rm num select)
  (let ((sel (new heap :of select
                  :for (new random :of '(1 2 3)))))
    (process repeat num
             for set1 = (next sel #t)
             for set2 = (next sel #t)
             for set3 = (rm-spectrum set1 set2 :spectrum ':note
                                     :remove-duplicates #t)
             ;; play set1 as chord
             each k in set1 
             output (new midi :time (now) :keynum k)
             ;; play set2 as chord .5 secs later
             each k in set2
             output (new midi :time (+ (now) .5) :keynum k)
             ;; strum ring modulated set 1 sec later
             each k in set3 as x from 0 by .1
             output (new midi :time (+ (now) 1 x) :keynum k)
             wait 3)))

(events (random-rm 15 (loop for i from 60 to 72 collect i))
        "test.mid")
⇒ "test.mid"

(events (random-rm 15 '(60 62 63 65 67 68 70 72))
        "test.mid")                   
⇒ "test.mid"
