[Function]
(hertz freq [:hz cool] [:in tuning] [:through scale] )

Returns the Hertz value of freq, which can be a note, keynum, Hertz value or list of the same.

hertz supports the following keyword parameters:

:hz bool
If true then a numerical freq is interpreted as a Hertz value rather than as a key number.
:in tuning
The tuning to use. Defaults to the global variable *scale*. The parameters :in and :through are mutually exclusive.
:through {tuning | mode}
Filters freq through tuning or mode and returns the hertz value of the closest integer keynum. The parameters :in and :through are mutually exclusive. 
cm> (hertz 'c4)
261.625
cm> (hertz 0)
8.175
cm> (hertz '(69 69.5 70))
(440.0 452.892 466.163)
cm>
