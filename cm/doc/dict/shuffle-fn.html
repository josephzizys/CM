[Function]
(shuffle list [:start i] [:end i] [:state x] [:copy b])

Returns a random reordering of list. Reordering takes place between start and end, which default to the full length of the sequence. If copy is true (the default) then a reordered copy of list is returned, otherwise list is destructively modified and returned. state is a random state object and defaults to *random-state*. 

cm> (shuffle '(a b c d e))
(b a e d c)
cm>
