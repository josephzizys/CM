[Function]
(note freq [:hz bool] [:in tuning] [:in? scale] [:through mode] [:from mode])

Returns the note name of freq, which can be a note name, key number, Hertz value or list of the same.

note supports the following keyword parameters:

:hz bool
If true then a numerical freq is interpreted as a Hertz value rather than as a key number.
:in tuning
The tuning to use for note references. Defaults to the global variable *scale*. The keywords :in, :in?, :through and :from are mutually exclusive parameters.
:in? {tuning | mode}
Tests if freq is a note reference in the specified tuning or mode. Returns that value or false. The keywords :in, :in?, :through and :from are mutually exclusive parameters.
:through mode
Filters freq through mode and returns mode's closest note in tuning. The keywords :in, :in?, :through and :from are mutually exclusive parameters.
:from mode
Converts a key number freq specified in mode's coordinates into a note name in the mode's tuning. The keywords :in, :in?, :through and :from are mutually exclusive parameters. 
cm> (note 440 :hz t)
a4
cm> (note 60)
c4
cm> (note 60 :accidental 's)
bs3
cm> (note 'bs3 :accidental 'ff)
dff4
cm> (note '(c4 d e1 f g5 ef f))
(c4 d4 e1 f1 g5 ef5 f5)
cm> (loop for k from 60 to 67
          collect (note k :accidental (list false 's)))
(c4 cs4 d4 ds4 e4 f4 fs4 g4)
cm>
