[Macro]
(new class {init value}*)

Allocates and initalizes a new object. class is the unquoted class name (symbol) of the type of object to create. Following class comes zero or more slot initializations where each initialization consists of a symbol or keyword slot init name followed by its value. 

cm> (new seq :name 'stuff)
#<seq: Stuff>
cm> (new midi :keynum (between 32 89) :time 1 :duration 3)
#i(midi time 1 duration 3 keynum 53 amplitude 64 channel 0)
cm> (new heap :rhythms '(q w e h e.. ))
#<heap @ x4C357A6>
cm>
