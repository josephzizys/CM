[Function]
(pickl list [:end n] [:start n] [:avoid x] [:state x])

Returns a randomly chosen element from list. If specified, start is a lower bound and end is the exclusive upper bound to selection from list. Use avoid to specify an element that cannot be directly reselected. state defaults to *random-state*. 

cm> (pickl '(1 two 3 four 5))
3
cm>
