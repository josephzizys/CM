[Macro]
(io object {name value}*)

Creates or reinitializes an event stream. 

cm> (io "/zap/test.aiff" )
#<clm-audio-stream: "/zap/test.aiff">
cm> (io "/zap/test.aiff" srate 44100 channels 2)
#<clm-audio-stream: "/zap/test.aiff">
cm>

Initializations for all streams:

:name string
The name of the file or port, including any directory information.

Initializations for all file streams:

:versions {number | true}
Causes a new version of the file to be created each time output occurs. The version name is based on the name of the file and the version count, which is initialized to number and incremented each time output occurs. A true value for versions is the same as 1.
