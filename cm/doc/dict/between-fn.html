[Function]
(between lb ub [exclude] [state])

Returns a random number equal to or greater than lb but less than ub. If both lb and ub are integers then an integer value is returned, otherwise a floating point value is returned.

between supports the following optional arguments:

exclude
Forces the return value to not equal exclude.
state
The random state object to use.

Examples:

Example 1. Random selection using between. 

(between 1 3)
⇒ 1
(between 1 2.0)
⇒ 1.234551
;; ten random integers without direct repetitions
(loop repeat 10 for n = (between 1 5 n) collect n)
⇒ (4 2 3 4 1 4 2 3 1 4)
