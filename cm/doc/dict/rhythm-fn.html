[Function]
(rhythm [tempo] [beat])

Returns the value of rhythm converted to seconds. rhythm can be a proportion (number), a rhythmic symbol or list of the same. tempo provides the metronome speed and defaults to the value of *tempo*, initially set to 60.0. beat specifies the metronome's pulse and defaults to the value of *beat*, initially set to .25 (quarter note.) 

cm> (rhythm 'q)
1.0
cm> (rhythm 1/4 120.0)
.5
cm> (rhythm 'w+q )
5.0
cm> (define *tempo* 120.0)

cm> (rhythm '(q th e.. x) )
(0.5 0.6666666666666666 0.4375 0.03125)
cm>
