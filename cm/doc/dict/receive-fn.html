[Function]
(receive stream . args)

Reads non-blocking stream and returns any input data. The format of this data as well as the avaliability of any args depends on the type of stream. This function is normally not needed but is useful when reading input from multiple ports at the same time, for example, when reading input from a pm-stream and an sc-stream inside a generic receiver.
