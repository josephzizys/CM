[Function]
(odds prob [true] [false] [state])

Returns true if a randomly generated number is less than prob, othewise false. The default values for true and false are boolean true and false, respectively. state is a random state object and defaults to *random-state* 

cm> (odds 1)
#t
cm> (odds 0 'ouch)
#f
cm> (odds .2 "Yup" "Nope")
"Nope"
cm>
