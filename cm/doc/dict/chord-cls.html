[Class]
chord

A specialized pattern that returns a list of chord members each time it is read by the next function. If chord data is a pattern then the next period of elements from the pattern will be returned as the chord members.

Examples:

Example 1. A process that plays mostly chords. The middle chord selects different chord members each time it is picked. 

(define (dochords)
  (process with pat = (new heap
                           :of (list
                                '(c4 e4 g4)
                                (new chord
                                     :of (new heap
                                              :of '(df4 ef4 gf4 af4
                                                        bf4 df5 ef5)
                                              :for 3))
                                '(e4 a4 cs5)
                                (new cycle :of '(bf6 bf2) :for 1)))
           repeat 32
           do (doeach (k (next pat))
                (output (new midi :keynum k :time (now))))
           wait .5))

(events (dochords) "test.mid")
⇒ "test.mid"
