[Class]
midi-pitch-bend

A MIDI pitch bend event.

midi-pitch-bend supports the following slot initializations:

:time number
Sets the start time of object in score.
:channel integer
A MIDI channel number. Defaults to 0.
:bend integer
A pitch bend value from -8192 to 8191. Defaults to 0, or no bend. See example below for more information.

Examples:

Example 1. Creating MIDI pitch bends. 

(new midi-pitch-bend :time 0 :channel 3 :bend -102) 

(define (step->bend val . width)
  ;; convert a step value +-width to bend range -8192 to 8191.
  ;; default width is +-2 (whole step up or down)
  (let ((wid (if (null? width) 2 (car width))))
    (inexact->exact
     (floor (rescale val (- wid) wid -8192 8191)))))

;;; bend one quarter-tone down

(new midi-pitch-bend :bend (step->bend -.5))

;;; specifying a percentage or 'cent' bend

(new midi-pitch-bend :bend (step->bend 75 100))
