[Function]
(plotter-data plotter {keyword value}*)

Returns the data sets loaded in plotter according to keyword arguments.

plotter-data supports the following keyword arguments:

:layer {bool :front :selection}
If boolean false (the default) then all the data sets are returned in a list of lists. Otherwise, if layer is boolean true or :front, then only the front layer's data is returned. If layer is :selection and there is a mouse selection then only those points in the front layer are returned otherwise all the front layer's data are returned.
:xy bool
Determines if CLOS objects or xy coordinate numbers are returned as data. If :xy is not specified then point data are converted to xy coordinate values but other CLOS instances are not. Otherwise, if :xy is specfied then its boolean value determines if xy coordinates or objects are returned. If xy coordinates are returned, the value of x-decimals and y-decimals determines their numerical format.
:x-decimals {bool | integer}
Specifies the number of decimal places to which x values values in :xy coordinate lists are rounded to (if :xy is true). If boolean true (the default) then x values are unchanged. If false then x values are floored to the nearest integer. Otherwise the value should the number of decimal places that data values are rounded to. Specifying zero causes data to remain floating point but without remainder.
:y-decimals {bool | integer}
Exactly like :x-decimals except that it applies to y values.
