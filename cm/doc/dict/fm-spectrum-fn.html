[Function]
(fm-spectrum spec1 spec2 {keyword value}*)

Performs ring modulation on the input spectra spec1 and spec2 and returns the resulting spectrum as a list of hertz values, key numbers or note names. Partials in the output spectrum are calculated as the difference and sum tones between pairwise combinations of the partials in the input spectra. Values in the input spectra can be specified as hertz values, key numbers or note names but both lists must be of the same type. Each input spectrum can be list of values or a single value.

fm-spectrum supports the keyword arguments:

:spectrum { :note | :keynum | :hertz | false}
Determines what type of values are returned in the output spectrum. The default value is false, which means that values in the output spectrum will be the same type as the input spectra.
:hz boolean
If true then values in the input spectra are Hertz values. The default value is false.
:remove-duplicates boolean
If true then the output spectrum will not contain any duplicate frequencies.
