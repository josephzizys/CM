Common Music Tutorials

  • intro.lisp, an overview of basic sound operations.
  • editing.lisp, an introduction to sequencing and editing operations.
  • scales.lisp, working with tunings and modes.
  • markov.lisp, an introduction to composing with Markov patterns.
  • jazz.lisp, sprouting processes, program changes, etc.

Most of these tutorials were initially prepared for the Tutorial on Music Composition in Lisp at the ILU Conference, Stanford University, June 2005.

Using the tutorials

Read and practice the tutorials in Emacs with Lisp syntax highlighting turned on (Options menu) and Common Music running in then Listener. Lisp expressions outside #| |# comment blocks is should be evaluated by clicking on the first or last parentheses in the expression and using the Listener's keyboard command C-x C-e to evaluate the expression in the Listener window.

The tutorials were written in Common Lisp syntax. To use the tutorials in Scheme evaluate the following expressions before running any examples:

Macro definitions for Scheme. 

;;; evaluate these before attempting examples in Scheme

(define-macro defun (name formals . body)
  `(define (,name ,@formals) ,@body))

(define-macro defparmeter (name value)
  `(define ,name ,value))
