Overview:
The Plotter Window

Not much to say at this point!

  • To add new points: Control-Mouse-<click>
  • To select point: Mouse-<click>
  • To add or remove points from selection: Control-Shift-Mouse-<click>
  • To move the selection: Mouse-<drag>
  • To lasso points inside a region: Mouse<sweep>

Exporting Data

Use File>Export to export the current layer or selection to the clipboard. Once exported use COMMAND-V to paste the data into the current Edit window.
