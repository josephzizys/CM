Overview:
The Grace Editor

The Grace text editor supports the following set of features:

  • Editing modes for SAL, Lisp and Text files
  • Syntax highlighting for SAL and Lisp programs
  • Syntactic line indentation (Tab key)
  • Code evaluation services
  • Emacs Lite command emulation

Editing Modes

The behavior of an Editor window depends on its "editing mode". The Grace editor currently supports three different modes: SAL, Lisp and Text. You choose the editing mode you want to work with when you create a new editing window with the File>New Editor command. When you open an existing file using File>Open the editing mode is selected automatically according to the file's extension:

SAL
.sal
Lisp
.lisp .lsp .cl .scm .cm .clm .ins .cmn .fms .asd
Text
.text .txt .*

Both SAL and Lisp editing modes provide syntax highlighting, or code coloring, based on the underlying mode's syntactic coding rules. Each editing mode also provides a line indentation service that you can use to help keep your programs readable. To use it simply press the Tab key to indent the current line according to the coding style rules of the underlying mode.

Both SAL and Lisp syntaxes provide code evaluation services:

  • To execute a SAL command, place the cursor at the end of the command and press COMMAND-Enter.
  • To evaluate a LISP expression, place the cursor at the end of the sexpr and type COMMAND-Enter.

Note: Lisp must be running in order to execute SAL or Lisp code. See Console Help for more information about starting Lisp.

Command Key Shortcuts

The Grace editor provides numerous Command key shortcuts for accessing common text editing operations.

  • On Linux and Windows the Command key is the Control key.
  • On OS X the Command key is the Apple key just to the left of the Spacebar.

On Linux and Windows the Command key shortcuts are disabled if Emacs Emulation is turned on. On OS X Emacs Mode and the Command key may be active at the same time.

Emacs Lite Commands

Use Options>Emacs Mode to enable/disable Emacs Mode in the the current edit window. On OS X the default setting for Emacs Mode is enabled, on Windows and Linux it is disabled. The Grace editor supports a fair number of Emacs editing commands. Emacs commands are keystroke combinations involving the CONTROL and/or META keys combined with one (or more) additional keys. In the documentation that follows pressing CONTROL or META is shortened to C- and M- respectively. For example, C-f means to hold down the Control key and type the f key and C-M-f means to hold down both the Control and Meta keys while typing the f key. Note: on OS X the Meta key is the Option key just right of the control key. On Linux and Windows the Meta key is the Alt key just to the left of the Spacebar.
