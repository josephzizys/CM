Changelog

Version 3.2.4

  • Enhancement: new Csound audio target. Does not require libcsound and should work everywhere. See csound documentation and the Help menu's csound example for more information.
  • Enhacement: added SndLib documentation.
  • Cleaned up numerous dictionary entries including a new entry for sprout. Dictionary now validates as xhtml strict.
  • system now builds on Windows/Mingw using Chicken 3.0.0. Carlos Pita reports that you can build with the latest Chicken (3.4.0) if you simply regenerate ChickenBridge.cpp. Instructions are in readme.text.
  • enhancement: Carlos Pita cleaned up the premake.lua script and added support for linking against installed systems. To link against an installed dependancy simply pass the install root as the value of the option eg '--juce /usr/local/'
  • enhancement: audio player now sports a realtime waveform view (lifted from the Juce Demo app...)
  • enhancement: new "Load..." menu item in Console and Editor works for both .scm and .sal files.
  • enhancement: new "Load Recent" menu holds last 8 loaded files for fast loading.
  • enhancement: new "Set Init File" menu item sets a lisp file to autoload at startup.
  • enhancement: added "Clear Init File" menu item for removing a lisp startup file.
  • bug fix: removed spurious AudioFile entry from Help menu and tidied up the menu.
